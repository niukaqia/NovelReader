能本地⼩说阅读器。
技术要点:
➢ Navigation 组件：实现书架→阅读器→设置⻚⾯的导航
➢ ⽂件操作使⽤鸿蒙⽂件管理 API 读取本地存储
➢ LazyForEach 优化书架列表渲染性能

##  核心功能
### 1. 书架管理
- 书籍展示 ：以卡片形式展示书籍封面和阅读进度
- 分类筛选 ：支持全部、小说、文学、科技、历史等分类
- 书籍扫描 ：自动扫描本地存储中的 TXT/EPUB 文件
- 阅读进度显示 ：格式为 第X章 第Y页
### 2. 阅读器核心
- 章节解析 ：自动识别章节标题（支持"第X章"、"第X回"等格式）
- 翻页功能 ：支持上一页/下一页，跨章节自动切换
- 阅读进度记忆 ：自动保存当前章节和页码，下次打开自动恢复
- 书签功能 ：支持添加、查看、跳转到书签位置
### 3. 阅读体验
- 主题切换 ：浅色、深色、护眼（sepia）、绿色四种主题
- 字体设置 ：支持字体大小（12px-32px）和行间距调整
- 翻页模式 ：滑动翻页、翻页效果、滚动模式
- 语音朗读 ：集成 Speech Kit 实现文本转语音

## 技术实现
### 数据持久化
- 使用 Preferences API 进行本地存储
- 书籍列表、阅读进度、书签分别存储
- 采用 单例模式 确保服务全局唯一
### 章节解析算法
- 使用 正则表达式 匹配章节标题
- 支持多种格式： 第1章 、 第一回 、 第一章
- 将书籍内容分割为多个 Chapter 对象
### 翻页机制
- 固定每页字数（默认1000字/页）
- 跨章节翻页：当前章节读完自动切换到下一章节
- 翻页时自动保存阅读进度
### 状态管理
- 使用 @State 、 @Prop 、 @Watch 装饰器管理组件状态
- aboutToAppear() 生命周期钩子进行初始化
- @Watch 监听属性变化实现响应式更新
- 阅读配置 ：字体、主题、亮度等个性化设置
- 书籍管理 ：清空书籍、重新扫描
- 配置持久化 ：所有设置自动保存到本地
